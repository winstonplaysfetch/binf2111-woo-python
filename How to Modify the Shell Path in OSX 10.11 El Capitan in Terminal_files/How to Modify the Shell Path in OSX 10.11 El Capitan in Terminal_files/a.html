                                                                                      Hunter Korgel Guest • 2 years ago

                                                                                      You might have figured this out already, but when you set PATH=$PATH:x:y:z the '$PATH:' simply means include the current value for PATH. In other words, you are just appending x, y, and z to the PATH that currently exists. If you want to delete a directory from PATH, simply leave it out when you define your PATH. You will have to write out all the directories, separated by colons, rather than use the shorthand $PATH.
